Detailed Description

Modulo che gestisce le strutture dati principali e le relative operazioni per la creazione, manipolazione, accesso e distruzione.

Questo file contiene l'implementazione concreta degli Abstract Data Type (ADT) dichiarati in dati.h: Animale, Lista, Nodo e NegozioData. Tutte le operazioni sui dati avvengono internamente, garantendo l'incapsulamento e fornendo un'interfaccia pubblica ben definita tramite i puntatori opachi.

Authors
Favale Giuseppe, Di Palma Marco, Gobbo Angelo
